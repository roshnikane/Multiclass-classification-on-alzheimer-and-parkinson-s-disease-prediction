# Multiclass Classification Disease Prediction 
Parkinson’s Disease affects mobility and speech in older adults, while Alzheimer’s impairs memory and cognition. Both are hard to detect early, causing misdiagnosis. We built a solution on that same kind of symptoms disease with a minor different detected the person having the parkinson's, alzheimer or the person is normal with the accuracy 90%-92%. This approach supports early detection and timely treatment.
